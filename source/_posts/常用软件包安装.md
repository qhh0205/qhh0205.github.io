常用软件包安装:
* dig命令安装
```bash 
 sudo yum install bind-utils
```
* pip一键安装
``` bash
 wget https://bootstrap.pypa.io/get-pip.py &&  sudo python get-pip.py && rm -f get-pip.py
```
* pstree命令安装
```bash
 sudo yum install psmisc
```
* python socks包安装
```bash
 sudo pip install PySocks
```
