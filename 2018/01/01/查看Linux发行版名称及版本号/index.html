查看Linux发行版名称及版本号

发表于 2018-01-01 | 分类于 Linux | 浏览 次

对于linx发行版及版本号的查看有如下几种方法，当一种方法失效的时候可以试试其他几种：

  • cat /etc/os-release

  • cat /etc/issue

  • lsb_release -a

  • cat /etc/redhat-release(针对redhat，Fedora)

  • cat /proc/version
